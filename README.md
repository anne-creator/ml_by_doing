# ml-by-doing

> Copy. Paste. Run. Learn ML by doing it.

You are curious about machine learning but have no idea where to start and no time to take a course. This repo is for you.

Each experiment is one script, one concept, 25 minutes. You run it in your browser for free. You see something real happen. Then you understand what just happened.

No setup. No math. No prerequisites.

## Experiments

| # | What you run | Concept | Time |
|---|---|---|---|
| [RL-01](./RL-01/) | Train a model on math problems | GRPO, agentic RL | 25 min |

More coming.

---

## How to use this

1. Pick an experiment from the table
2. Open the folder, read the tutorial
3. Follow the steps — everything runs in Google Colab, free, in your browser

That is it.

---

## What you need

Nothing installed. Just a Google account.

If you want to run locally: Python 3.11+, at least 16GB RAM. Apple Silicon works. MacBook Pro with 48GB runs 7B model experiments without any cloud GPU.

---

## Who this is for

Developers who can write code but have never touched ML. You do not need to understand the math to start. The understanding comes from running it.
